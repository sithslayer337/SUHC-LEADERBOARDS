# SUHC Leaderboards — All Time

A clean, dark-themed leaderboard site for Survival UHC stats. Hosted on GitHub Pages.

## Features

- Track players across **CIV**, **HG**, and **UHC** game modes
- Sort by any stat column
- 6 rank tiers: Recruit → Soldier → Veteran → Elite → Champion → Legend
- Click any player for a detailed stat breakdown
- Add new players via the form
- Data persists in your browser via localStorage
